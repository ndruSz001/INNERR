# research_china

Esta sección contiene módulos para la ingesta, análisis y conexión de información sobre robótica y desarrollo orientado a China.

## Estructura
- robotics_china.py: Investigación y conexión de datos sobre robótica en China.
- research_online.py: Ingesta y conexión de información online sobre páginas chinas y temas de interés.

## Uso
1. Agrega fuentes relevantes usando `add_source(url, description)`.
2. Obtén la información más reciente con `fetch_latest()`.

## Expansión
Preparado para integrar scraping, APIs, y análisis avanzado.
