# Base de Datos Biomédica Personal

Aquí puedes guardar y organizar información relevante para tu investigación biomédica: papers, datasets, resultados, contactos, protocolos, y notas clave.

## Sugerencias de organización
- `papers/` — Artículos científicos y resúmenes.
- `datasets/` — Datos experimentales y públicos.
- `protocols/` — Protocolos y procedimientos.
- `contacts/` — Colaboradores y expertos.
- `results/` — Resultados y análisis.
- `notes/` — Notas rápidas y aprendizajes.

Puedes crear estas carpetas según lo necesites para mantener tu información biomédica organizada y accesible.
