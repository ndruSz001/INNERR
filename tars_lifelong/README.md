# TARS Lifelong AI

Estructura base para el asistente de vida profesional, aprendizaje continuo y gestión personal.

## Carpetas y archivos
- `knowledge/` — Base de conocimiento, papers, resúmenes, aprendizajes.
- `projects/` — Seguimiento de proyectos, ideas, experimentos.
- `skills/` — Habilidades, cursos, competencias y planes de mejora.
- `config/` — Configuración, preferencias, seguridad y privacidad.
- `logs/` — Registro de actividades, hitos y reflexiones.
- `datasets/` — Datasets personales o de interés para entrenamiento.
- `scripts/` — Scripts de automatización, entrenamiento y análisis.

## Integración
Esta estructura se integra con los módulos existentes de TARS y puede crecer según tus necesidades.
